# Dark Mode (High Contrast Black) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a High Contrast Black dark mode (system-preference default, persisted nav-bar toggle) to the 98.css-styled Astro blog, per `docs/superpowers/specs/2026-06-11-dark-mode-design.md`.

**Architecture:** All Win98 colors move into CSS custom properties in a new `src/styles/theme.css` (`:root` = current light values, `html[data-theme="dark"]` = HC Black). Component/page styles swap hardcoded hexes for `var()` (specificity unchanged). A pre-paint inline script in `BaseHead.astro` resolves the theme; a nav button toggles and persists it; utterances comments sync via dynamic injection + `postMessage`.

**Tech Stack:** Astro 6, 98.css v0.1.21 (static `/98.css`, not edited), vanilla JS, no test framework (verification = `pnpm build` + browser checks + grep assertions).

**Files:**

- [ ] **Step 1: Insert meta + inline script.**

Replace:
```html
<meta name="generator" content={Astro.generator} />

<!-- 98.css Framework -->
```
With:
```html
<meta name="generator" content={Astro.generator} />

<!-- 主题初始化：在样式表加载前确定 data-theme，避免错误主题闪烁（FOUC） -->
<meta name="theme-color" content="#c0c0c0" />
<script is:inline>
	(function () {
		var theme = null;
		try {
			theme = localStorage.getItem('theme');
		} catch (e) {}
		if (theme !== 'light' && theme !== 'dark') {
			theme = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
		}
		document.documentElement.dataset.theme = theme;
		var meta = document.querySelector('meta[name="theme-color"]');
		if (meta) meta.setAttribute('content', theme === 'dark' ? '#000000' : '#c0c0c0');
	})();
</script>

<!-- 98.css Framework -->
```

`is:inline` keeps Astro from bundling/deferring it; running inside `<head>` guarantees it executes before first paint.

Run: `pnpm build && grep -c "prefers-color-scheme" dist/index.html`
Expected: `1` (script inlined into every page; spot-check `dist/en/index.html` likewise).

In the dev-server browser: `localStorage.setItem('theme','dark'); location.reload()` → `document.documentElement.dataset.theme` is `"dark"` with no light flash on reload. `localStorage.removeItem('theme'); location.reload()` → follows OS setting.
